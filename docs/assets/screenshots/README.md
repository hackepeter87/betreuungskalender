# Documentation screenshots

This directory contains screenshots made exclusively with fictional
demonstration data.

Before replacing or adding an image, follow the privacy checklist in
[`docs/screenshots.md`](../../screenshots.md). Do not add real names, addresses,
notes, evidence references, private domains, authentication details, or
calendar data from a private case.

Recommended filenames:

- `dashboard-desktop.png`
- `calendar-desktop.png`
- `entry-mobile.png`
- `reports-desktop.png`
