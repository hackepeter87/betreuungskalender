# v0.1.0 - Initial public preview

This is the first public preview of Betreuungskalender.

## Included

- Child management
- Care entries and a mobile agenda
- Planned-versus-actual contact schedule
- Holiday periods and unavailable periods
- Trips and costs
- Monthly, quarterly, and annual analytics
- JSON export and import
- Separate CSV exports
- Neutral PDF report and A4 print view
- Plausibility and data-quality checks
- Field help and documentation rules
- Monthly closing and audit history
- Responsive iPhone and iPad layouts with PWA foundations
- Fastify and SQLite API with migrations and security configuration
- SQLite backup and restore tooling
- Container, LXC, reverse-proxy, and authentication documentation
- GitHub Actions CI with a Vite build
- Sensitive-artifact release check

## Important notes

This application is a documentation tool. It does not provide legal advice or
replace review by a qualified lawyer.

PDF and print reports are technical summaries of documented data. They are not
official or legally reviewed documents.

The usefulness of reports depends on the completeness and accuracy of the
entered data.

At the time of the v0.1.0 tag, the browser UI and SQLite API used separate data
stores. Current `main` has since made SQLite/API the source of truth and added
a guided migration path for legacy browser data.
