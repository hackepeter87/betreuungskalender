## Runtime smoke test

The archive was extracted into an isolated temporary directory and installed
with production dependencies:

```bash
npm ci --omit=dev
```

Result: passed.

The published runtime was started from the extracted archive with a temporary
SQLite database and backup directory:

```bash
NODE_ENV=production \
HOST=127.0.0.1 \
PORT=34130 \
DATABASE_PATH=/tmp/betreuungskalender-v1.3.0-runtime-data/app.sqlite \
BACKUP_DIR=/tmp/betreuungskalender-v1.3.0-runtime-backups \
LOG_LEVEL=warn \
```

The health endpoint reported:
